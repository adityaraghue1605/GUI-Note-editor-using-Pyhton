# GUI-Note-editor-using-Pyhton
# 📝 Simple Text Editor (Python Tkinter)

A basic desktop Text Editor built using Python and Tkinter.

## 🚀 Features

- Create New File
- Open Existing .txt Files
- Save Files
- Menu Bar with File Options
- Clean & Minimal GUI

## 🛠️ Technologies Used

- Python 3
- Tkinter (Built-in GUI Library)
